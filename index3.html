A set of GSA tools for Galaxy

$39

Galaxy-GSA is a group of GSA tools wrapped in a Galaxy platform. Each Galaxy-GSA tool gives visibility to all the software options, parameters, and help from the original packages, so the user can leverage Galaxy’s simple and reproducible infrastructure to upload data, format data, choose parameters and options through a simple interface, and send results to the history to allow comparisons between different runs, communication between tools, and communication with all other Galaxy tools.

Educational material to learn GSA

$49

GSA Classroom is a repository of slides, videos, and animations, to learn Gene Set Analysis.

A database of GSA methods and tools

$49

GSARefDB is the most comprehensive database of GSA methods and tools. It is available as an excel file or as a shiny app, and contains information regarding the publication, website (if exists), software tool (if exists), and an index of each paper's popularity. It is also a database of GSA reviews and papers applied to specific fields such as genomic regions, ncRNAs, and others.

Tools for benchmarking GSA methods

$49

GSA BenchmarKING is a collection of tools to automatically benchmark groups of GSA methods. Benchmarking tools include jupyter notebooks and shiny apps.
